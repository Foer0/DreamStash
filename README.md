# Dream Stash

Это full-stack веб-приложение, объединяющее современный пользовательский интерфейс и надежный серверный API. Проект полностью контейнеризирован с помощью Docker, что обеспечивает быстрый и предсказуемый запуск в любой среде.

## Технологический стек

* **Бэкенд:** Python, FastAPI, SQLAlchemy, Alembic
* **Фронтенд:** Vue.js, Vite
* **База данных:** PostgreSQL
* **Инфраструктура:** Docker, Docker Compose

## Быстрый запуск (Локально)

Для запуска проекта на вашем компьютере должен быть установлен [Docker Desktop](https://www.docker.com/products/docker-desktop/).

### 1. Клонирование репозитория
```bash
git clone <ссылка-на-твой-репозиторий>
cd <название-папки-репозитория>
```

### 2. Настройка переменных окружения
В корневом каталоге проекта создайте файл `.env`. В нем нужно указать пароль для базы данных:
```env
POSTGRES_PASSWORD=ваш_надежный_пароль
```

### 3. Запуск контейнеров
Выполните команду для сборки и запуска всех сервисов:
```bash
docker compose up -d --build
```

### 4. Применение миграций БД
Чтобы создать необходимые таблицы в базе данных, выполните миграции внутри контейнера бэкенда:
```bash
docker compose exec backend alembic upgrade head
```

## Доступ к сервисам

После успешного запуска сервисы будут доступны по следующим адресам:

* **Фронтенд (Vue):** [http://localhost:5173](http://localhost:5173)
* **API Документация (Swagger):** [http://localhost:8000/docs](http://localhost:8000/docs)
* **База данных:** `localhost:5432` (пользователь: `postgres`, база: `dream_stash`)

## Остановка проекта

Чтобы остановить контейнеры без потери данных в БД:
```bash
docker compose down
```

## Структура проекта

* `/backend` — Исходный код API на FastAPI и настройки базы данных.
* `/frontend` — Пользовательский интерфейс на Vue.js.
* `docker-compose.yml` — Конфигурация развертывания.
